# Categorio

Categorio game at lefun.fun.

Still a work in progress!


## Prerequisites

* [Node 20+][node]
* [pnpm 9+][pnpm]


## Quick commands

### Install the dependencies

    make install

### Start de dev server

    make dev

### Build the packages

    make build

### Format the code base

    make format

### Run the tests

    make test


[node]: https://nodejs.org
[pnpm]: https://pnpm.io
